# 🛒 E-Commerce Management System (MySQL Project)

This project is a comprehensive **E-Commerce Database Management System** built using **MySQL**. It includes multiple interlinked tables to simulate real-world e-commerce operations and supports complex analytical queries using **Joins, Subqueries, Aggregate Functions, Date Functions, Grouping, and Filtering**.

---

## 📁 Project Overview

- **Database Name:** `Riddhi`
- **Language Used:** SQL (MySQL)
- **Project Type:** Academic / Practice-based DBMS Project
- **Tables Created:** `Categories`, `Products`, `Customers`, `Orders`, `OrderDetails`, `Shipping`, `Discounts`, `Reviews`, `Coupons`

---

## 🔧 Features & Requirements

✅ 10+ Table Relationships  
✅ 40+ Analytical SQL Queries  
✅ Inner, Left, Right Joins  
✅ Subqueries  
✅ Group By + Having  
✅ Date Functions  
✅ Aggregate Functions (SUM, COUNT, AVG, etc.)  
✅ Includes at least 70–100 test records
